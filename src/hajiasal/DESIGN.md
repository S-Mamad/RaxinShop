# حاجی عسل — Design System

**Direction:** Dark Luxury Editorial  
**Audience:** خریدار premium فارسی، اعتمادمحور

## Palette

| Token | Value | Usage |
|---|---|---|
| `--void` | `#0a0a0a` | پس‌زمینه صفحه |
| `--surface` | `#141414` | سکشن / کارت |
| `--surface-elevated` | `#1c1c1c` | سطح بالاتر |
| `--text-primary` | `#f5f5f4` | متن اصلی |
| `--text-secondary` | `#a8a29e` | متن ثانویه |
| `--gold` | `#d4a056` | CTA، لینک کلیدی |
| `--gold-bright` | `#e8b86a` | hover CTA |

## Typography

- Body: Vazirmatn 400/500/600
- Display: Lalezar
- h1: max 2–3 خط، `text-wrap: balance`
- قیمت: `tabular-nums`

## Rules

- استورفرانت تاریک لوکس؛ بدون تم روشن cream
- Icons: Phosphor
- بدون Unsplash؛ تصاویر از `/images/hajiasal/...`
- مسیرها فقط با `hajiasalPath()`
- موبایل‌اول: فیلتر فروشگاه bottom-sheet، منوی کشویی، sticky CTA در PDP
- بدون em-dash، بدون Inter، بدون بنفش AI

## Admin

- پنل ادمین می‌تواند روشن/خنثی بماند (جدا از فروشگاه)
